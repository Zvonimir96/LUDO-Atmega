# About
The goal of this project was to create an electronic version of the board game LUDO in order to promote interest in the STEM field. The initial concept was first implemented in Python to validate the core game mechanics, control logic, and overall design. The playable software prototype is available for download in the [Git repository](https://github.com/Zvonimir96/LUDO-Python).

After the software solution was successfully developed, the focus shifted to the hardware design. The complete hardware development process, including schematics and implementation details, is also documented in the [Git repository](https://github.com/jkordek1/Man-Don-t-Get-Angry-Board-game). The project ultimately integrates both software and hardware components, resulting in a fully functional electronic LUDO game platform.
